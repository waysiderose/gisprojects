Custom Basemap Using Google

Step 1: Choose an organization.

This exercise operates under customizing a basemap to the colors of an organization. Previously, we had used ArcGIS Esri software to do this. However, what if our client does not have a license?

I chose the non-profit Global Links. They do good work that has local and far reaching impact - for example, they are a primary mask supplier in a volunteer effort I help out in



Step 2: Create a color pallet.

To get a reference image, I took a screen shot of the main page of Global Links.


Then, I ran the screenshot through Adobe Color Create. Color Create pulls colors from a source and give Hex codes as output. I used the "Extract Theme" option, which produces multiple sets of five colors from an image source based on a "theme". For example, there are "colorful", "deep", and "dark" options, which change the color output. I tested chose multiple colors from different themes that worked together in order to create a reference table.


Created a color pallet reference table with a Column for the Hex Code, Color Name, Sample Image, and RBG Distribution.
